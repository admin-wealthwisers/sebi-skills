## Installation

This repo is a Claude plugin marketplace. To install:

### In Claude Desktop

1. Open Claude Desktop → **Settings** → **Plugins**
2. Switch to the **Personal** tab
3. Click **Add marketplace**
4. Paste this URL: `https://github.com/admin-wealthwisers/sebi-skills`
5. Click **Sync**
6. Once the marketplace appears, install the `sebi-algo-trading-2026` plugin from within it.

### In Claude Code

```bash
/plugin marketplace add <your-handle>/sebi-skills
/plugin install sebi-algo-trading-2026
```

Once installed, the skill loads automatically when you discuss Indian
algorithmic trading, broker APIs, strategy classification, or related
compliance topics. No need to invoke it explicitly.
